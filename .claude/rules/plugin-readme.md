# プラグインREADME作成ガイドライン

プラグインのREADME.mdはユーザーがプラグインを理解し、利用するための重要なドキュメントです。

## 必須セクション

以下の3セクションは必須です（日本語/英語どちらでも可）:

| セクション | 説明 |
|-----------|------|
| `## 概要` または `## Overview` | プラグインが何をするか |
| `## インストール` または `## Installation` | インストール手順 |
| `## 使い方` または `## Usage` | 基本的な使用方法 |

## 推奨構造

```markdown
# プラグイン名

## 概要

このプラグインは〜を実現します。

## インストール

\`\`\`bash
claude plugin install example-plugin
\`\`\`

## 使い方

\`\`\`bash
# 基本的な使い方
/example-command arg1 arg2
\`\`\`

## 設定（オプション）

設定項目がある場合に記載。

## ライセンス（オプション）

MITなど。
```

## コードブロック

コードブロックには必ず言語を指定:

```markdown
\`\`\`bash
echo "良い例"
\`\`\`

\`\`\`
echo "悪い例（言語指定なし）"
\`\`\`
```

## リンク

- 相対パスリンクは実在するファイルを参照すること
- 画像も同様に実在するファイルを参照すること
- 外部URLは検証対象外

```markdown
<!-- 良い例 -->
詳細は[ガイド](docs/guide.md)を参照。
![スクリーンショット](images/screenshot.png)

<!-- 悪い例（ファイルが存在しない） -->
詳細は[ガイド](docs/nonexistent.md)を参照。
```

## 避けるべきこと

- 必須セクションの省略
- 言語指定のないコードブロック
- 存在しないファイルへのリンク
- 過度に長い説明（簡潔に）
