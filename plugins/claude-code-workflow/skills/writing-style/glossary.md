# 表記揺れ辞書

統一表記を定義する。AIが文章を書く際にもこの辞書を参照する。

## 固有名詞・サービス名

| 正 | 誤・揺れ | 備考 |
|---|---|---|
| Claude Code | claude code, ClaudeCode | 公式表記に準拠 |

## 略称ルール

| 正 | 誤・揺れ | 備考 |
|---|---|---|
| リポジトリ | リポ | 「GitHubリポ」「Privateリポ」等の略称は避ける |
| GitHubリポジトリ | GitHubリポ | 同上 |
| Privateリポジトリ | Privateリポ | 同上 |
| モノレポ | （略称のまま使用可） | 一般的な技術用語として定着しているためそのまま使う |

## 禁止表現（AI 特有 / 自分の語彙にない表現）

| 避ける表現 | 代替案 | 備考 |
|---|---|---|
| ——（em ダッシュ） | 「。」で文を切る、または括弧・「、」で繋ぐ | AI 特有の書き方。自分の文体ではない |
| 刺さる | 合っている / よかった / 使える | AI が好んで多用する。自分の語彙にない |
| 効く | 合っている / よかった / 使える | 同上 |
| 〜することができます | 〜できます | 冗長。AI 特有の言い回し |

## 煽り・誇張表現

| 避ける表現 | 代替案 | 備考 |
|---|---|---|
| 絶対やるべき | 〜すると楽 / 〜が便利 | 煽り表現は避ける |
| 知らないとヤバい | 〜を知っておくと役立つ | 同上 |
| マジで | （削除） | 同上 |
